# node更新
1. 安装nvm
```bash
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
```
2. 加载nvm命令
```bash
$ source ~/.bashrc
```

3. 查看已经安装的node
```bash
$ nvm ls
```
![](/assets/img/nvm_ls.png)

4. 查看远程可安装的node版本
```bash
$ nvm ls-remote
```
![](/assets/img/nvm_ls_remote.png)

5. 选择对应版本安装
```bash
$ nvm install 19.8.1
```
![](/assets/img/node_update.png)
